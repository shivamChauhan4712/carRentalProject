# carRentalProject(NAME-RoadRunner Car Rental System)
 ```
**RoadRunner Car Rental System**

A user-friendly web application for seamless car rental. Key features include:

* Easy booking of shared 5-seater or 7-seater cars.
* Faster booking confirmations and direct driver communication.
* Real-time tracking of car availability and driver location.
* Efficient car fleet management for optimal operations.

Built using Java, Spring Boot, and MySQL.
```
